# 🖼️ Image Gallery

A responsive and interactive **Image Gallery** built using **HTML, CSS, and JavaScript**.  
This project displays images in a clean grid layout with hover effects, category filters, and a lightbox popup with next/previous navigation.

---

## 📌 Features

- Responsive image gallery layout
- Hover zoom effect
- Smooth transitions
- Lightbox popup view
- Next / Previous image navigation
- Category filter buttons
- Mobile-friendly design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
